# Aidoku Website

Website for [Aidoku](https://github.com/Aidoku/Aidoku). Public site located at https://aidoku.app/.

## Building

This project uses [zola](https://github.com/getzola/zola) for building a static site. Install zola and run `zola build` to build the entire site, or `zola serve` for live-reloading changes during development.
