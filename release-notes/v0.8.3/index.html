v0.8.3 Latest

May 18, 2026

Main Changes

  • New migration flow

    View series details and optionally copy data instead of completely migrating.

    Commit
  • Kavita API key login
  • Continuous with gaps reading mode

    Like the webtoon mode, but with gaps between pages.

    Commit
  • Fixed tracker search bar not showing
  • Page offset setting and double page improvements (@chibijoam)

Other Features

  • Resume last opened chapter setting (@phonixdrive)

    Option to always select the most recently opened chapter rather than the next chapter in the list for the continue reading button.

    Commit
  • Display Kavita chapter titles
  • Include built-in sources in backups
  • Apple pencil squeeze actions

    Single squeeze to move to the next page, double squeeze for previous, and long squeeze for chapter selector.

    Commit
  • Disable double tap zoom option

    Makes tap zone presses activate faster if enabled.

    Commit
  • Simplified layout setting

    Turns the "manga per row" settings into standard, compact, and custom options.

    Commit
  • Confirmation when removing tracked item from library
  • Don't save source filter changes when in Incognito Mode
  • Various improvements to menus

    Updated icons and ordering to match system apps, use large buttons for downloading and sharing chapters.

    Commit
  • Refresh series details menu option

    Alternative to pulling to refresh inside a series details page.

    Commit
  • View on website option in tracker search
  • Tracker chapter number offset option (@bcmeireles)

    Set an offset number to apply to whatever chapter number you read and gets sent to the tracker when updating automatically.

    Commit

Other Fixes

  • Relative paths in Kavita base URL not working
  • Incorrect Kavita and Komga source keys

    Could've caused issues when restoring data from backups.

    Commit
  • Read button not working after filtering chapters
  • Chapter scanlator filter sometimes clearing in library refreshes
  • Some settings not working on iOS 18
  • Various text reader issues (@rbertus2000)
  • Shikimori tracker not working

    Updated domain to one with less region locking.

    Commit
  • Add to category prompt sometimes not showing
  • Library/category re-locking when scrolling upwards
